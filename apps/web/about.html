About Wave Finder

Wave Finder is a free, open source web app helping spearfishers and divers in Port Elizabeth plan their dives with confidence.

Created and maintained by Joshua Vlantis, Wave Finder is built for the local community and welcomes contributions. View the project on GitHub.

Have questions or feedback? Email admin@wavefinder.org.
